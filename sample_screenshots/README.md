# Sample screenshots — Torama Vote

A place to keep real app screenshots for promotional / marketing use (website,
social, app stores, decks). Drop PNG/JPG files here.

> Note: screenshots shared in chat can't be copied into the repo automatically —
> save them into this folder from your device/computer.

## Suggested shots to capture (and good captions)

| File (suggested) | Screen | Caption idea |
|------------------|--------|--------------|
| `results-leaderboard-live.png` | Public results | "Live results — watch the count update in real time." |
| `ballot-candidates.png` | Voter ballot | "Vote in seconds. Candidate cards with photos & bios." |

## Tips for clean promo shots
- Use a phone/browser with a clean status bar (or crop it).
- Prefer an election with real candidate names + photos and a few votes in.
- Light theme renders consistently (the app declares `color-scheme: light`).
- Capture the live leaderboard while an election is **open** with **Live results**
  to show the pulsing "Live" badge.
